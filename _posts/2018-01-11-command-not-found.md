`xxxx : command not found`
1 路径PATH不对
<br>sudo root 普通用户PATH不同
<br>添加PATH解决
2 未按照相关程序
<br>apt install 安装
3 命令不存在
4 脚本为centos写的,命令和ubuntu不一样
5 Shell不对
`#!/bin/sh`和`#!/bin/bash`能运行的命令不一样
